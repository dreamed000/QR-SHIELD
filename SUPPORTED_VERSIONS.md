# Supported Versions

## Python

QR-SHIELD supports:

- Python 3.10

- Python 3.11

- Python 3.12

- Python 3.13

## Browser

- Firefox (current stable releases supported)

- Firefox Extended Support Release (ESR) where available

## Dependencies

- Selenium 4.45.x

- urllib3 2.7.0

- requests 2.34.2

- Pillow 12.3.0

- Jinja2 3.1.6

- user-agent 0.1.14

## Supported Environments

QR-SHIELD is tested on:

- Windows 10/11

- Linux distributions with Firefox installed

- WSL2 with Firefox available on the host system

## Notes

- Platform updates for target services can affect functionality.

- This project is research-focused and may require maintenance for compatibility with upstream changes.
