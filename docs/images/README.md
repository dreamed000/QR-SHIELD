# Documentation Images

This directory contains images used in documentation.

## Image Categories

### Screenshots

- CLI interface screenshots

- Browser automation examples

- Module output examples

### Diagrams

- Architecture diagrams

- Data flow diagrams

- Module workflow diagrams

### Badges

- Status badges

- License badges

- Build status badges

## Adding Images

1. Place image files in this directory

1. Use descriptive filenames

1. Include alt text in markdown

1. Keep file sizes reasonable (<1MB)

## Supported Formats

- `.png` - Screenshots, diagrams

- `.svg` - Vector diagrams

- `.jpg` - Photographs (rarely used)

- `.gif` - Animated examples

## Image Guidelines

- Clear and readable

- Descriptive captions

- Appropriate alt text

- High contrast for accessibility

- Minimal file size
