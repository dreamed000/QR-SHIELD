## Open Source Projects

QR-SHIELD builds on the excellent work of these open-source projects:

### Core Dependencies

**Selenium**

- Browser automation framework

- License: Apache 2.0

- GitHub: [github.com/SeleniumHQ/selenium](https://github.com/SeleniumHQ/selenium)

- Without Selenium, QR-SHIELD's browser automation wouldn't be possible.

**Jinja2**

- Template rendering engine

- License: BSD

- GitHub: [github.com/pallets/jinja](https://github.com/pallets/jinja)

- Powers dynamic HTML rendering for phishing templates.

**Pillow (PIL)**

- Image processing library

- License: HPND/PIL

- GitHub: [github.com/python-pillow/Pillow](https://github.com/python-pillow/Pillow)

- Essential for QR code image extraction and processing.

**requests**

- HTTP library

- License: Apache 2.0

- GitHub: [github.com/psf/requests](https://github.com/psf/requests)

- Simplifies HTTP operations and session handling.

**urllib3**

- HTTP client library

- License: MIT

- GitHub: [github.com/urllib3/urllib3](https://github.com/urllib3/urllib3)

- Underlying connection pooling and HTTP capabilities.

**user-agent**

- User agent generator

- License: Apache 2.0

- PyPI: [user-agent package](https://pypi.org/project/user-agent/)

- Provides realistic user agent rotation for requests.
